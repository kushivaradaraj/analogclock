# Analog Clock - Dark/Light

This project is a web-based analog clock with additional features such as light and dark themes, real-time display, and a stopwatch mode.

## Table of Contents
- [Introduction](#introduction)
- [Features](#features)
- [Controls](#controls)

## Introduction
This project provides a simple and customizable analog clock that runs in the browser. It includes a visually appealing clock display with the ability to switch between light and dark themes. Additionally, the clock supports a stopwatch mode, providing a versatile user experience.

## Features
1. **Real-time Analog Clock:**
   - Displays the current time with hour, minute, and second hands.

2. **Stopwatch Functionality:**
   - Allows users to toggle between the analog clock and stopwatch modes.

3. **Light and Dark Themes:**
   - Enhances user experience with the option to switch between light and dark themes.

## Controls
- **Start/Stop Button:**
  - Initiates or halts the clock's real-time updates.
  
- **Toggle Mode Button:**
  - Switches between Analog Clock and Stopwatch modes.
